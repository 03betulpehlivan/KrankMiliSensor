Crankshaft Position Sensor (CPS) Demo

This project is a comprehensive study that examines the working principles, hardware architecture, and fault detection procedures of the Crankshaft Position Sensor (CPS)—a key component considered the heart of internal combustion engines—from a technical perspective. 📄

About the Project

Modern automotive electronics rely heavily on sensor technologies for engine timing and synchronization. In this study:

The physical structure of the sensor
Signal generation mechanisms
Different sensor types (Magnetic vs. Hall Effect) are analyzed comparatively
Technical Content and Highlights 🛠
General Description

The interaction between the Engine Control Unit (ECU) and the electromechanical component that detects crankshaft rotational speed (RPM) and angular position.

Working Principle

Position calculation based on the electromagnetic induction law (Faraday’s Law) using a “60-2 tooth” wheel structure.

Hardware Architecture
Iron core
Permanent magnet
Coil windings
Sensor Comparison
Magnetic (Inductive): Durable, low cost, analog signal output
Hall Effect: High precision, digital square-wave output, fast response without ADC requirement
Technical Specifications
Operating temperature: $-40^{\circ}C$ to $+150^{\circ}C$
Air gap sensitivity: 0.5–1.5 mm
Measurement capacity: 10,000+ RPM
Fault Diagnosis
Signal inspection using diagnostic tools (DTC analysis)
Signal validation using oscilloscope-based testing

# Krank Mili Konum Sensörü (Crankshaft Position Sensor - CPS) Demo

Bu proje, içten yanmalı motorların kalbi sayılan **Krank Mili Konum Sensörü (CPS)**'nin çalışma prensiplerini, donanımsal mimarisini ve arıza tespit prosedürlerini teknik bir perspektifle ele alan kapsamlı bir çalışmadır. 📄

---

## Proje Hakkında
Modern otomotiv elektroniğinde motor zamanlaması ve senkronizasyonu için kritik öneme sahip sensör teknolojileri detaylandırılmıştır. Bu çalışmada:  

- Sensörün fiziksel yapısı  
- Sinyal üretim mekanizmaları  
- Farklı sensör tipleri (**Manyetik vs. Hall Efekt**) karşılaştırmalı olarak analiz edilmiştir  

---

## Teknik İçerik ve Öne Çıkanlar 🛠

### Genel Tanım
Krank milinin dönüş hızını (RPM) ve açısal pozisyonunu algılayan elektromekanik bileşenin, motor kontrol ünitesi (ECU) ile olan etkileşimi.

### Çalışma Prensibi
Faraday'ın Elektromanyetik İndüksiyon Yasası temelinde "60-2 dişli" yapısı ile konum hesaplama.

### Donanımsal Mimari
- Demir nüve  
- Kalıcı mıknatıs  
- Bobin sargıları  

### Sensör Karşılaştırması
- **Manyetik (Endüktif):** Dayanıklı, maliyeti düşük, analog sinyal üretimi  
- **Hall Efekt:** Hassas, dijital kare dalga çıkışı, ADC gerektirmeyen hızlı tepki süresi  

### Teknik Spesifikasyonlar
- Çalışma sıcaklığı: $-40^{\circ}C$ ile $+150^{\circ}C$  
- Hava boşluğu (air gap) hassasiyeti: 0.5–1.5 mm  
- Ölçüm kapasitesi: 10.000+ RPM  

### Arıza Teşhisi
- Diagnos cihazı (DTC analizi) ile sinyal kontrolü  
- Osiloskop yardımıyla sinyal doğrulama süreçleri
